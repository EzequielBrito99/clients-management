# Clients Management System

## Description

The Clients Management System is a web-based application designed to help businesses efficiently manage their client information. This project provides features such as list, adding and editing client records, as well as viewing detailed client profiles. It is built with modern web technologies to ensure scalability, performance, and ease of use.

## Features

- List, add and edit client records.
- View detailed client profiles.
- Search and filter clients.
- Responsive design for mobile and desktop devices.

## Technologies Used

- React (v17/v18) with TypeScript.
- Material UI (v4) for high-quality UI/UX components.
- React Router Dom for seamless SPA navigation.
- Context API for global state management (Auth & Notifications).
- Axios for optimized API communication.

## Installation

Follow these steps to set up the project locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/ezequielbrito99/clients-management.git
   ```
2. Navigate to the project directory:
   ```bash
   cd clients-management
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm start
   ```

## Usage

1. Open your browser and navigate to `http://localhost:3000` (or the specified port).
2. Use the interface to manage client records:
   - List clients.
   - Add new clients.
   - Edit existing client information.
   - Search and filter through the client list.
